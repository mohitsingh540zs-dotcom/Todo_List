# React ToDo List App

A simple and responsive ToDo List application built with **React** and **Vite**. This project demonstrates the fundamentals of React, including components, state management, props, event handling, and dynamic rendering.

---

## Features Completed

* React project setup using Vite
* Functional Components
* Header component
* ToDo List component
* ToDo Item component
* Controlled input field
* Add new tasks
* Dynamic rendering using `map()`
* Unique key for each task
* Responsive table layout for displaying tasks
* Tailwind CSS styling

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd todo-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`) in your browser.
